# petshop
애완동물 간식, 사료 등 판매 웹사이트 제작

[client 실행] <br>
cd client <br>
npm install <br>
npm start <br>
(ID: root, PW: 1234)

[server 실행] <br>
cd server <br>
npm install (port 8080) <br>
.env 파일 MYSQL 관련 데이터 수정 <br>
(MYSQL_USER, MYSQL_PASS, MYSQL_CHARSET) <br>
npm start <br>

[MySQL] <br>
올려놓은 Data import (파일명: database.sql)
